MYFLIX: MOVIE API

Obj: To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

REST API Breakdown

Description URL (+parameters) HTTP Method Request Data Format Response Data Format
1. Return a list of ALL movies to the user /movies GET None JSON object containing data of all movies
2. Return data (ID, description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user. /movies/[title] GET None JSON object containing data on a single movie (Title, Genre, etc). Example: {id:1, Title: "The Lion King", Genre: Children, Director: Disney, etc..}
3. Return data about a single movie by genre /movies/genre/[genreTitle] GET None JSON object containing data about genre of single movie. Example: {name: 'Comedy', Description: 'Makes you laugh and go hehe haha'}
4. Return data about a director (bio, birth year, death year) by name /movies/[director]/[name] GET None JSON object containing data about director. Example: {name: 'Roger Allers', Birth Year: '1949', Death Year: 'N/A', Bio: 'Roger Charles Allers is an American film director, screenwriter, animator, storyboard artist, and playwright. He is most well known for co-directing The Lion King'}
5. Allow new users to register /users POST JSON object containing user data. Example: {username: '...', password: '...', email:'...@.com'} JSON object containing user data added along with an ID and Top Movies list. Example: {username: '...', password: '...', email:'...@.com', topMovies: [...], id:'...'}
6. Allow users to update their username /users/[username] PUT None JSON object containing updated user data. Example: {username: '...', password: '...', email:'...@.com', topMovies: [...], id:'...'}
7. Allow users to add a movie to their list of favorites. /users/[username]/[topMovies] POST None Message confirming movie has been added
8. Allow users to remove a movie from their list of favorites /users/[username]/[topMovies] DELETE None Message confirming movie has been deleted
9. Allow existing users to unregister. /users/[username] DELETE None Message confirming user has unregistered.

Uses:
